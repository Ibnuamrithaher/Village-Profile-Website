# 🏛️ Website Profil Desa Petung

Website profil desa yang modern, interaktif, dan premium untuk **Desa Petung**, Kecamatan Dongko, Kabupaten Trenggalek. Dibangun menggunakan teknologi web terbaru untuk memberikan pengalaman pengguna yang luar biasa serta informasi publik yang transparan.

## ✨ Fitur Utama
- 💎 **Desain Modern**: Menggunakan konsep *Glassmorphism* yang elegan dan bersih.
- 📱 **Fully Responsive**: Tampilan optimal di semua perangkat (Mobile, Tablet, Desktop).
- 🎨 **Animasi Premium**: Transisi halus dan efek *scroll-reveal* menggunakan Framer Motion.
- 🧭 **Navigasi Cerdas**: Sticky navbar dengan indikator halaman aktif.
- 📦 **Layanan Publik**: Informasi lengkap mengenai layanan mandiri, ADM penduduk, dan transparansi dana desa.
- 🔗 **Integrasi Data Resmi**: Aset (logo & foto infrastruktur) bersumber langsung dari Website Resmi Kabupaten Trenggalek.

## 🚀 Teknologi yang Digunakan
- **React 19** - Library UI yang kuat.
- **Vite** - Build tool generasi terbaru yang sangat cepat.
- **Tailwind CSS v4** - Framework styling utility-first terbaru.
- **Framer Motion** - Library animasi tingkat lanjut.
- **Lucide React** - Set ikon yang konsisten dan modern.
- **React Router Dom** - Sistem navigasi antar halaman.

## 🛠️ Cara Menjalankan Project

### 1. Kloning Repositori
```bash
git clone https://github.com/Ibnuamrithaher/Village-Profile-Website.git
cd Village-Profile-Website
```

### 2. Instalasi Dependencies
```bash
npm install
```

### 3. Jalankan Mode Development
```bash
npm run dev
```

### 4. Build untuk Produksi
```bash
npm run build
```

## 📂 Struktur Folder
```text
src/
├── assets/          # Aset gambar & aset visual

├── pages/           # Halaman Utama (Home, About, Services, Contact)
```
